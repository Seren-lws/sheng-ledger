# 声声记账本

一款为个人定制的 PWA 记账应用，核心目标：**记得快、看得清、AI 帮你算**。

技术栈：Next.js + Supabase + Vercel 部署，PWA 形式使用，移动端优先。

---

## 功能总览

### 记一笔（首页）
快速记账入口，支持收入/支出切换。

- 自定义数字键盘，支持简单加减运算（如 150+200）
- 日期快选：今天 / 昨天 / 前天，也可打开日历
- 选择支付账户后币种自动确定
- 提交后对应账户余额自动增减
- 标签多选，按使用频率排序，支持快速新建/删除
- 记账成功后 Toast 提示

### 明细
按日期分组的流水时间线。

- 多维筛选：分类、标签、账户、日期范围、金额范围
- 支持搜索备注和标签
- 点击查看详情，可编辑/删除

### 固定支出
每月"还没花就已经没了的钱"一目了然。

- 新增 / 编辑 / 删除固定支出项（名称、金额、币种、支付账户、扣款日、备注）
- 固定支出月度总额 & 各项占比
- 固定支出 vs 总支出比例
- 按支付渠道分组查看

### 存钱理财
看清楚"我到底存下了多少"。

- 月度净存款 & 存款率自动计算
- 近 6 个月存款趋势图
- 可自由支配存款 = 总资产 - 下月固定支出 - 预估生活费（近 3 月均值）
- 环比对比（vs 上月）
- 存钱目标：设定目标金额，显示进度条和预计达成时间
- 近 3 月月均存款统计

### 数据分析 + AI
不只是图表，还有一个懂你财务状况的 AI 助手。

- 月度收支趋势（柱状图）
- 支出分类占比（环形图 + 明细）
- 收入结构分析
- 支付渠道使用统计
- AI 对话：接 Claude API（中转站 yunwu.ai），可以问"这个月花了多少""存款比例多少""外食是不是太多了"

### 我的
个人设置与资产管理中心。

- 账户管理：新建/编辑/删除账户（名称、币种、余额），查看关联流水
- 总资产概览：按基准货币显示，JPY ↔ CNY 一键切换
- 分币种小计
- 汇率手动设定（格式：100 JPY = X CNY）
- 数据导出：CSV 格式，支持本月/上月/近 3 月/全部，中文 BOM 头兼容 Excel
- 记账提醒：Web Notification，默认 21:00 检查当天是否记账，时间可自定义

### 多币种
- 支持 JPY 和 CNY
- 币种跟随账户，记账时无需手动选
- 汇率全局可见（ExchangeRateBanner 组件）
- 总览和分析中基准货币可切换

### PWA
- manifest.json 配置完整（声声记账本 / 记账本）
- Service Worker 离线缓存
- 应用图标（算盘风格，莫兰迪配色）
- 支持添加到主屏幕

## 设计规范

- **色彩**：莫兰迪彩虹色系（玫红 #C4A09B、珊瑚 #C9A88A、暖沙 #CBBEA6、橄榄 #A3AB8E、鼠尾草 #9FB5A3、薄荷 #8FB5B0、天青 #94ACBE、薰衣草 #ADA8C6、灰粉 #B89EB5）

- **卡片**：纯白 #FFFFFF，16px 圆角
- **风格**：温暖柔和，大圆角、柔色调、充足留白
- **布局**：移动端优先，底部 5 个 Tab 导航（记一笔 / 明细 / 存钱 / 分析 / 我的）

## 本地开发

```bash
# 安装依赖
npm install

# 复制环境变量并填入
cp .env.local.example .env.local

# 启动开发服务器
npm run dev
```

### 环境变量

```
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
AI_API_KEY=your_yunwu_api_key
AI_BASE_URL=https://yunwu.ai/v1
AI_MODEL=claude-sonnet-4-20250514
```

## 页面结构

```
📱 底部导航栏（5 个 tab）：

[ 记一笔 ]  [ 明细 ]  [ 存钱 ]  [ 分析 ]  [ 我的 ]

/                → 记一笔（快速记账）
/transactions    → 明细（流水筛选）
/savings         → 存钱理财（存款追踪 + 目标）
/analysis        → 分析（图表 + AI 对话）
/profile         → 我的（资产 / 设置 / 导出）
/fixed-expenses  → 固定支出（从「我的」进入）
```
