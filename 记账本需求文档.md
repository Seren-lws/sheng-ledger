# 💰 晚声记账本 — 需求文档

## 项目概述

一款为个人定制的 PWA 记账应用，替代鲨鱼记账。核心目标：**记得快、看得清、AI帮你算**。

技术栈：React (Next.js) + Supabase + Vercel 部署，PWA 形式使用。

---

## 一、核心模块

### 1. 记一笔（首页）

记账的主入口，追求最少操作步数。

- 支持记录 **收入 / 支出**
- 每笔记录包含：金额、分类、标签（可多个）、支付账户、备注（可选）、日期
- **智能输入**：
  - 输入"全家 饭团 咖啡 350"→ 自动识别金额 350、分类→餐饮/便利店、标签→饭团+咖啡、商家→全家
  - 历史词联想：输入时弹出以前用过的标签/商家，直接点选
  - 常用标签优先展示，不常用的收起来
- 选择支付账户后，**币种自动确定**（跟着账户走）
- 提交后，对应账户余额自动增减
- **数字键盘**（参考鲨鱼记账）：
  - 右侧第四列功能键：日期选择、加号（+）、减号（-）
  - 日期快选：点击日期键弹出快捷面板，支持"今天/昨天/前天"一键选择，也可打开日历选更早日期
  - 支持简单加减运算（如 150+200）

### 2. 资产管理

管理所有"钱包"，随时掌握钱都在哪里。

- 自建账户，每个账户绑定：名称、币种（JPY/CNY）、当前余额
- 示例账户：みずほ銀行（JPY）、paypay（JPY）、微信钱包（CNY）、支付宝（CNY）、现金-日元（JPY）
- 总资产概览：
  - 按基准货币显示总额，**基准货币可切换**（日元 ↔ 人民币）
  - 分币种小计：日元资产 xx 万円 / 人民币资产 xx 元
- **汇率手动设定**：用户自行输入当前汇率（如 1 CNY = 20.5 JPY），用于换算总览，随时可更新
- 每个账户可查看关联的收支流水

### 3. 固定支出管理

每月"还没花就已经没了的钱"一目了然。

- 独立页面，新增/编辑/删除固定支出项
- 每项包含：名称、金额、币种、支付账户（渠道）、固定扣款日期（月/日）、备注
- 示例：房租 6.5 万円/月（银行卡）、Spotify 980円/月（信用卡）、iCloud 6元/月（支付宝）
- 数据展示：
  - 固定支出月度总额
  - 各项占比
  - 固定支出 vs 总支出的比例
  - 按支付渠道分组查看

### 4. 存钱理财

把手写记录自动化，看清楚"我到底存下了多少"。

- **月度存款追踪**：
  - 每月净存款 = 当月总收入 - 当月总支出（自动计算）
  - 存款率 = 净存款 / 总收入（百分比）
  - 月度存款趋势图：每个月存了多少，是涨还是跌

- **可用资产 vs 待花资产**：
  - 总资产中，扣除下个月预计固定支出和预估生活费后，剩余多少是"真正存住的钱"
  - 计算公式：可自由支配存款 = 总资产 - 下月固定支出 - 预估生活费
  - 预估生活费基于近 3 个月非固定支出平均值，自动计算

- **存款增长追踪**：
  - 和上月比：存款是增加了还是减少了？增减了多少？
  - 和去年同期比（数据积累后）
  - 累计存款增长曲线

- **存款速度 & 预测**：
  - 近 3/6 个月平均每月存款额
  - 按当前速度预测：N 个月后总存款可达多少
  - 支持设定存款目标（如"年底存到 xx"），显示进度和达标概率

- **AI 洞察**（融入分析模块）：
  - "这个月存了多少？比上月多还是少？"
  - "按目前速度，年底能存多少？"
  - "我的存款率在什么水平？有没有优化空间？"

### 5. 数据分析 + AI

不只是图表，而是一个懂你财务状况的助手。

#### 基础图表
- 月度收支趋势（折线图/柱状图）
- 支出分类占比（饼图）
- 收入结构分析
- 各支付渠道使用频率和金额

#### AI 分析（接 Claude API，中转站 yunwu.ai）
- 对话式交互，可以问问题：
  - "这个月花了多少？比上个月多在哪？"
  - "我的存款比例是多少？"
  - "最近外食是不是太多了？"
  - "按现在的花法，年底能存多少？"
- AI 能读取账本数据进行计算和总结
- 可主动提供月度/季度复盘摘要

## 二、分类与标签体系

### 分类
- 用户自定义，只保留常用的，精简为主
- 预设一套初始分类，用户可增删改：
  - 支出：餐饮、交通、购物、住房、娱乐、医疗、日用、AI、其他
  - 收入：工资、日语课时费、写作收入、采耳收入、礼物
- 每个分类分配一个**莫兰迪彩虹色**，颜色在分类创建时指定或自动分配

### 标签
- 与分类独立，每笔可挂多个标签（多选）
- 预设初始标签：早饭、午饭、晚饭、电车、api充值、咖啡、全家、罗森
- 标签管理：记账页内可直接添加、删除、编辑标签
- 标签自由创建，从历史输入中自动积累
- 常用标签置顶，支持搜索
- 标签统一使用中性灰底色，选中时跟随当前分类颜色高亮

---

## 三、多币种方案

- 币种跟随账户，记账时选账户即确定币种，无需手动选
- 支持 JPY 和 CNY 两种货币
- 汇率由用户手动维护（设置页填入，如 1 CNY = xx JPY）
- **汇率全局可见**：记一笔、明细、存钱、我的页面顶部或合适位置显示当前汇率（如「1 CNY = 20.5 JPY」），方便随时确认和快捷修改
- 总览和分析中，基准货币可一键切换：
  - 看日元视角：所有人民币资产/支出按汇率折算为日元
  - 看人民币视角：所有日元资产/支出按汇率折算为人民币

---

## 四、预算功能

- 初期不设手动预算，等数据积累后自动生成
- 基于历史数据（如近 3 个月平均），自动建议每月预算
- 记账时如果当月支出接近/超过预算，给出提醒
- 后期可支持分类预算（如餐饮预算 xx 元/月）

## 五、UI 设计规范

### 整体风格
- 温暖柔和：大圆角、柔色调、充足留白
- 移动端优先，单手操作友好

### 色彩方案：莫兰迪彩虹色系

| 莫兰迪玫红 | #C4A09B | 餐饮 |
| 莫兰迪珊瑚 | #C9A88A | 交通 |
| 莫兰迪暖沙 | #CBBEA6 | 住房 |
| 莫兰迪橄榄 | #A3AB8E | 日用 |
| 莫兰迪鼠尾草 | #9FB5A3 | 购物 |
| 莫兰迪薄荷 | #8FB5B0 | 娱乐 |
| 莫兰迪天青 | #94ACBE | 医疗 |
| 莫兰迪薰衣草 | #ADA8C6 | AI |

- 标签：统一使用中性灰调底色，不带分类颜色
- 账户图标：各账户可自选莫兰迪色
- 背景色：暖白 #FAF8F5
- 卡片：纯白 #FFFFFF，大圆角（16px），极细边框

## 七、页面结构

```
📱 底部导航栏（5个tab）：

[ 记一笔 ]  [ 明细 ]  [ 存钱 ]  [ 分析 ]  [ 我的 ]

记一笔 — 快速记账入口
明细   — 按时间查看流水，支持搜索/筛选（按分类、标签、账户、日期范围、金额范围）
存钱   — 存款追踪、可用资产、增长趋势、目标预测
分析   — 图表 + AI 对话
我的   — 资产管理、固定支出、分类管理、汇率设置、预算设置、数据导出
```

## 八、其他功能

- **数据导出**：支持导出 Excel/CSV
- **记账提醒**：每天晚上推送"今天还没记账哦"（PWA 通知）
- **多设备同步**：Supabase 天然支持
- **暗色模式**：不需要
